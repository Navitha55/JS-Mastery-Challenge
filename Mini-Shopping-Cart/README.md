# Day 1 - Mini Shopping Cart Simulation 🛒

## Objective

Build a JavaScript program that simulates a shopping cart system, calculates billing details, applies discounts and taxes, and demonstrates JavaScript fundamentals.

## Concepts Practiced

### Variables

* `let`
* `const`
* `var`

### Data Types

* String
* Number
* Boolean
* Array
* Object

### Scope

* Global Scope
* Function Scope

### Operators

* Arithmetic Operators
* Comparison Operators
* Assignment Operators

### JavaScript Runtime

* `setTimeout()`
* Delayed order confirmation simulation

## Features

* Display available products
* Add products to cart
* Check product availability
* Calculate cart total
* Apply tax
* Apply discount for eligible orders
* Generate final bill
* Simulate order processing and confirmation

## Learning Outcomes

Through this project, I learned:

* How arrays and objects are used to model real-world data
* How functions help organize code into reusable components
* How billing calculations are performed using operators
* The difference between global and function scope
* How `setTimeout()` works in JavaScript's runtime environment

## Sample Flow

```text
Display Products
        ↓
Add Available Products To Cart
        ↓
Calculate Total
        ↓
Apply Tax & Discount
        ↓
Generate Bill
        ↓
Processing Order...
        ↓
Order Confirmed
```
