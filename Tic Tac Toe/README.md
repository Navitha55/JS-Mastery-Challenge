# 🎮 Tic-Tac-Toe Game

A simple and interactive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**. This project helped me strengthen my understanding of core JavaScript concepts while building a complete browser-based game.

## 🚀 Features

* Two-player Tic-Tac-Toe gameplay
* Alternate turns between X and O
* Automatic winner detection
* Winner announcement popup
* Reset Game functionality
* New Game functionality
* Responsive design using Flexbox
* Modern dark-themed user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 Winning Logic

The game stores all possible winning patterns in an array and checks them after every move. If a player matches any winning pattern, a winner message is displayed and the board is disabled.

## ▶️ How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open the project folder

```bash
cd tic-tac-toe
```

3. Open `index.html` in your browser

No additional installation is required.
