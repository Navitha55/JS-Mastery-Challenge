# 🔎 Day 5 - Smart Search & Logger

## Objective

Build a smart search application that simulates searching a database while demonstrating modern JavaScript concepts such as throttling, debouncing, promises, promise chaining, and function currying.

## Features

- 🔍 Search through a simulated dataset
- ⌨️ Log user keystrokes using **throttling**
- ⏳ Trigger search only after typing stops using **debouncing**
- 🤝 Simulate database fetching using **Promises**
- 🔗 Format search results using **Promise Chaining**
- 📝 Generate structured logs using **Function Currying**
- 📄 Display matching search results in the browser

## Concepts Practiced

- Throttling
- Debouncing
- Promises
- Promise Chaining
- Function Currying
- Event Listeners
- DOM Manipulation
- Array Methods
