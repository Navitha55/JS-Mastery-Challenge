# Day 9 ✊✋✌️ Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built using **HTML, CSS, and JavaScript**. Players compete against the computer, with scores updated dynamically after each round.

## 🚀 Features

* Play Rock, Paper, or Scissors against the computer
* Random computer move generation
* Real-time score tracking
* Instant win, lose, or draw messages
* Dynamic message color updates based on game result
* Responsive and clean user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 Game Logic

The computer randomly selects one of the following options:

```text
Rock
Paper
Scissors
```

The game then compares the player's choice with the computer's choice and determines whether the player:

* Wins
* Loses
* Draws

Scores are updated immediately after each round.
