# Day 3 - Employee Bonus Calculator 💰

## Objective

Build a web-based Employee Bonus Calculator to practice modern JavaScript concepts such as Call, Apply, Bind, Array Methods, Spread Operator, and ES6 features.

## Features

- Display employee details
- Filter eligible employees based on salary
- Calculate bonus using `call()`
- Create updated employee objects using spread operator
- Calculate total bonus using `reduce()`
- Display bonus details for each eligible employee

## Concepts Practiced

- Call, Apply, Bind
- Array Methods (`filter`, `map`, `reduce`)
- Rest & Spread Operators
- Arrow Functions
- Destructuring
- Template Literals
- Default Parameters
- `let` and `const`
