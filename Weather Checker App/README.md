# 🌦️ Day 6 - Weather Checker App

## Objective

Build a Weather Checker web application that fetches real-time weather information using the OpenWeather API.

## Features

- 🌍 Search weather by city name
- 🌡️ Display current temperature
- ☁️ Display weather description
- ❌ Handle empty input validation
- ⚠️ Display user-friendly error messages for invalid cities
- 🔒 Safely access API data using Optional Chaining (`?.`)
- 🛡️ Handle API and runtime errors using `try...catch`

## Concepts Practiced

- Fetch API (GET Request)
- Async/Await
- Optional Chaining (`?.`)
- Error Handling (`try...catch`)
- HTTP Response Handling
- DOM Manipulation
- Event Listeners
