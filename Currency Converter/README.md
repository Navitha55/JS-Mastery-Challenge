# 💱 Day 7 - Currency Converter

## Objective

Build a Currency Converter web application that fetches real-time exchange rates and converts one currency into another.

## Features

- 🌍 Convert between different world currencies
- 💰 Enter a custom amount for conversion
- 🚩 Automatically update country flags based on selected currencies
- 🔄 Fetch live exchange rates using a public Currency API
- 🧮 Display converted amount instantly
- 🚀 Load the default exchange rate when the page opens

## Concepts Practiced

- Fetch API (`GET` Request)
- Async/Await
- DOM Manipulation
- Event Listeners
- Dynamic Dropdowns
- Template Literals
- Object Traversal
- Dynamic Image Updates
